TraceMyTarget is a Python-based tool designed for digital security analysis of domains, emails, and usernames. This tool was developed with the assistance of ChatGPT AI.

Key Features

Retrieves WHOIS data for domains and checks domain age

Verifies whether WHOIS information is private

Tracks IP addresses and flags high-risk countries

Extracts public activity traces on GitHub

Generates findings reports with risk levels: suspicious, medium, low

Advantages

Fast and easy to use

Compact and interactive

Provides clear risk indicators

Limitations

Works only with publicly available data

Does not fully detect personal data breaches

Analysis results depend on WHOIS availability and other public information

Built with Python and utilizes the following libraries: whois, ipwhois, requests, email_validator, BeautifulSoup4.
